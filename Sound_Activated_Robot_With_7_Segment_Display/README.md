🔊 Sound Activated Robot with 7-Segment Display 

A sound-activated robotic system that responds to external sound input (such as a clap) to initiate movement. The robot uses a sound sensor to detect sound signals and activates its motors for a short duration while displaying its status on a 7-segment display.

⚙️ How It Works

The sound sensor continuously monitors the environment for sound.

🔊 Sound Detected (HIGH signal):

The robot is activated, and both motors turn ON to move the robot forward.

⏸️ After Activation:

The robot continues running briefly and then can stop or wait for the next sound input.

🔢 Display Output:

1 → Robot Active (Moving)

0 → Robot Inactive (Stopped)

The 7-segment display provides real-time visual feedback of the robot’s state.

🔹 Features

Sound-based robot activation

Instant response to clap or noise

Visual feedback using 7-segment display

Simple and interactive control system

Easy to implement and expand

🧩 Components Used

Arduino Uno

Sound Sensor Module

4-Digit 7-Segment Display

Motor Driver (L293D)

DC Motors

Chassis & Wheels

Power Supply
