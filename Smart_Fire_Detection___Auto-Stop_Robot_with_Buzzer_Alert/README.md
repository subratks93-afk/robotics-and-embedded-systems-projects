🔥 Smart Fire Detection & Auto-Stop Robot with Buzzer Alert 

An autonomous safety robot designed to detect fire in real time and respond immediately by stopping movement and triggering an audible alarm. The system uses a flame sensor to monitor fire presence and an Arduino Uno to control robot motion and buzzer alerts through a motor driver.

⚙️ How It Works

The flame sensor continuously monitors the environment for fire.

🔥 Fire Detected (LOW signal):

The robot immediately stops, and the buzzer is activated to alert nearby users.

✅ No Fire Detected (HIGH signal):

The robot continues moving forward, and the buzzer remains off.

The system also provides real-time status updates via the Serial Monitor.

🔹 Features

Real-time fire detection using flame sensor

Automatic robot stop on fire detection

Instant buzzer alert system

Continuous forward movement during safe conditions

Fast response time with minimal delay

🧩 Components Used

Arduino Uno

Flame Sensor

Buzzer
